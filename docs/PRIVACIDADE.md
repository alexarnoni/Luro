# Política de Privacidade

## Dados coletados
- E-mail para autenticação por link mágico.
- Dados financeiros inseridos manualmente: contas, transações, metas, categorias, regras e insights.

## Uso dos dados
- Exclusivamente para funcionamento do aplicativo: autenticação, dashboards, gráficos e geração de insights financeiros.

## Armazenamento
- Banco de dados hospedado em servidor de provedor de nuvem.

## Compartilhamento
- Não vendemos ou compartilhamos dados com terceiros para fins comerciais.

## Direitos do usuário
- Você pode solicitar a exclusão da conta e de todos os dados associados a qualquer momento pela opção “Excluir minha conta”. A exclusão encerra a sessão e remove os registros financeiros associados.
